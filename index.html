Tạo thư mục và file cùng 1 lúc
  • - copy bỏ vào hết là chạy được
    New-Item -Path "F:\Project\README.md" -ItemType File -Force
    New-Item -Path "F:\Project\src\index.js" -ItemType File -Force
    New-Item -Path "F:\Project\assets\logo.png" -ItemType File -Force
Di chuyển một file Move-Item -Path "MyFolder\f5\file2.md" -Destination "MyFolder"
  • Lưu ý: nếu mà lấy file từ folder f5 di chuyển ra folder Myfolder thì phải đúng từ folder nhận file cần di chuyển là folder MyFolder
Di chuyển tất cả file trong một thư mục Move-Item -Path "C:\Source\*" -Destination "C:\Target\"
Di chuyển nhiều file cùng lúc Move-Item -Path "C:\Source\file1.txt", "C:\Source\file2.txt" -Destination "C:\Target\"
Di chuyển file với điều kiện cụ thể Move-Item -Path "C:\Source\*.txt" -Destination "C:\Target\"
Những cách nắm bắt thư mục và chỉnh thư mục làm sao
  1. | MyFolder
    | -- f5
    | | -- file1.md
    | | -- file2.md
    | -- f6
    | | -- file4.md
    | file3.md
  2. * Những điều cần chú ý:
       - Để ý đường dẫn trong terminal(powerShell), nếu mình đang làm việc với MyFolder mà đường dẫn lại là
       F:\Myfolder> thì lúc này phải chỉnh lại đường dẫn hoặc là dùng -path để chỉnh lại đường dẫn đúng(F:\MyFolder\file1.md)
       - Nếu mình đang muốn làm việc với con của MyFolder thì mình phải có tk cha lớn hơn đó là ổ đĩa F:\
